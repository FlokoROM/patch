# FlokoROM/patch

## Usage

普通に [crDroidAndroid/android](https://github.com/crdroidandroid/android/tree/13.0) を引っ張ってきたら、後は `floko.sh` の通りにすればパッチが当たります。

ロゴ画像を含むリポジトリについてのみ `patch` ではなく `git diff/apply` でやっていますが、後者に統一しても良い気はします。
